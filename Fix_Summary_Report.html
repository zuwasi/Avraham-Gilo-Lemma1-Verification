2. Domain Bound — P ∈ [0, 1/q), not [0, 1]

❌ Wrong: P → 1  →  ✅ Correct: P → 1/q
The model requires 1 − qP > 0, so P < 1/q. As P → 1/q, the term (q−1)/(1−qP) diverges to +∞, making N > 0 and dα*/dP < 0 unambiguously. This is a simpler and more rigorous argument than the P → 1 limit used in the original proof.

3. Uniqueness — IVT Gives ≥1 Peak, Not Single Peak

❌ Wrong: "By continuity, ∃ P̄ maximizing α*"  →  ✅ Correct: Three conditions required
The Intermediate Value Theorem only proves at least one zero crossing. Single-peakedness requires: (a) N < 0 near P = 0, (b) N > 0 near P = 1/q, and (c) N(P) strictly increasing — ensuring exactly one crossing.

4. Individual vs. Aggregate Demand

⚠ Missing: Aggregation step not addressed  →  ✅ Added: Sufficient condition stated
Individual single-peakedness does NOT automatically imply aggregate single-peakedness. A sufficient condition: if dα*/dP is strictly decreasing in P for all consumer types (regardless of how the types are distributed), then aggregate demand inherits single-peakedness.
